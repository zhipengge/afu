阿福（Afu）隐私政策

更新日期：2026年4月12日

一、适用范围
本隐私政策适用于阿福（Afu）iOS 应用，以及应用在本地设备上对聊天、图片、文件、模型下载与配置数据的处理方式说明。

二、我们处理的数据
1. 聊天内容
- 用户输入的文本消息
- 模型生成的回复内容
- 会话标题、时间等本地会话元数据

2. 用户主动提供的附件
- 用户从相册选择或拍摄的图片
- 用户通过“文件”选择器导入的 PDF、Word、Excel、CSV、TXT、Markdown 等文件
- 为便于展示与解析，应用可能生成缩略图、文本摘录或视觉摘要

3. 本地配置数据
- 用户设置的生成参数，例如温度、Top-P、最大回复长度、自定义系统提示词
- 模型下载偏好，例如是否优先使用官方源

4. 凭证信息
- 用户主动填写的 Hugging Face Access Token
- 该 Token 仅用于访问需要授权的模型下载地址

三、数据如何被使用
1. 聊天与推理
聊天内容、图片摘要、文件摘录会被用于构建提示词，以便本地模型在设备上生成回复。

2. 图片与文件解析
- 图片可能使用 Apple 提供的本机能力（如 Vision）进行文字识别、分类、人脸检测或画面摘要
- 文件可能在本机被解析为可供模型理解的文本摘要或节选内容

3. 模型下载
应用会请求远程模型清单，并根据用户选择从 Hugging Face 官方源或镜像源下载模型文件。

四、数据存储位置
1. 聊天记录
聊天记录及相关元数据默认存储在用户设备本地数据库中。

2. 附件摘要与缓存
图片视觉摘要、文件摘录、缩略图等可能保存在本地，以便提升多轮对话与界面展示体验。

3. Token 存储
Hugging Face Token 存储在 iOS 系统 Keychain 中，而非明文写入普通配置文件。

五、网络请求说明
阿福（Afu）默认以本地处理为主，不会将聊天内容主动上传到开发者自有业务服务器。

应用中可能发生的网络请求包括：
1. 拉取远程模型列表（models.json）
2. 从 Hugging Face 官方源或镜像源下载模型文件
3. 访问 gated 模型对应的 Hugging Face 页面（仅当用户主动点击）

除上述场景外，当前版本不以云端接口形式转发用户聊天内容用于远程推理。

六、第三方服务与系统能力
1. Apple 系统能力
应用可能调用 iOS 提供的相机、相册、文件选择、Vision、PDFKit、Keychain 等系统框架处理用户数据，这些处理通常发生在本机。

2. Hugging Face
当用户下载模型时，模型文件请求会直接发送至 Hugging Face 官方站点或其镜像站点；如模型属于 gated 类型，用户需自行提供 Token 并接受相关模型协议。

七、权限使用说明
1. 相机权限
用于拍照并将图片作为聊天输入。

2. 相册读取权限
用于从相册选择图片作为聊天输入。

3. 相册写入权限
用于将应用生成的分享卡片或图片保存到系统相册。

八、数据控制与用户权利
用户可自行：
1. 删除单条或全部对话记录
2. 删除已下载模型文件
3. 删除已保存的 Hugging Face Token
4. 停止继续使用应用，并删除本地应用数据

请注意：删除应用通常会移除应用沙盒中的本地数据，但系统级备份、设备同步策略等仍受 Apple 平台机制影响。

九、数据安全
我们尽量采用符合平台规范的方式保护本地数据，例如：
- 使用 iOS Keychain 存储敏感 Token
- 使用本地数据库保存会话内容
- 尽量减少不必要的远程传输

但任何软件系统都无法保证绝对安全。用户应妥善保管自己的设备访问权限与第三方 Token。

十、未成年人说明
如未成年人使用本应用，应在监护人指导下阅读并理解本隐私政策后再使用相关功能。

十一、政策更新
如果本隐私政策发生更新，我们会通过代码仓库、发布说明或应用相关文档更新版本内容。

十二、联系与说明
若你正在使用开源或内部测试版本，而仓库中尚未补充正式联系信息、官网或隐私政策页面，则以本文件作为当前版本的隐私处理说明。

如未来提供正式官网、邮箱或产品页，应以届时发布的正式版本说明为准。
